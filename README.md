# Gerenciador de Arquivos

Projeto em Java para gerenciar arquivos e pastas.

## Funcionalidades
- Criar arquivos
- Excluir arquivos
- Listar diretórios
- Organizar pastas

## Tecnologias
- Java

## Como executar
1. Compile o projeto:
   javac Main.java

2. Execute:
   java Main
